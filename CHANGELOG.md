# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

## [0.2.0] - 2024-04-02

### Added
- GitHub Actions CI/CD pipeline
- PyPI publishing support
- Pytest test suite
- Codecov integration for coverage reporting
- Multiple AI provider support (OpenAI, NVIDIA, Gemini, Ollama)
- Custom prompts functionality
- Audio output support
- System tray icon
- Settings dialog for configuration
- Cross-platform support (Windows, Linux, macOS, WSL)
- Dark theme UI with font resizing

### Changed
- Improved code detection heuristics
- Better error handling in clipboard operations
- Enhanced streaming response handling

### Testing
- Added unit tests for config module
- Added unit tests for AI providers
- Added unit tests for clipboard monitor
- Added Pyproject validation tests

## [0.1.0] - Initial Release

- Initial implementation
- Basic clipboard monitoring
- Code explanation popup
- Ollama integration
- Settings configuration
